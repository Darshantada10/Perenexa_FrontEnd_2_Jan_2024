Form

form tag it will help us to send user data to the server

there are 2 methods to send data to the server

  1. GET
  2. POST

Name attribute will help us to identify the data

GET Method

it will use url to send data that is why it is less secure

it has limitation of only 2048 characters

it does not support file uploading

get method will reload the page with previous data

? is called query string

& multiple data

faster than post method it is almost 90% faster we will use get method for search

POST Method

it will not use url to send data so it is more secure

No limitation

Supports file uploading

data will not be there when page is realoaded

slower than get method
